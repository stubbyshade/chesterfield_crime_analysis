## Analysis of Chesterfield Crime v1.0

This project was undertaken during the first semester of MSc Data Science at the University of Sheffield.
It allows the user to view specific crime rates, evolution of crime over time and to map where specific crimes take place.
The code currently contains analysis specific to Derbyshire Constabulary and the town of Chesterfield, but can be used to analyse other areas.

## Requirements

R Studio or other appropriate IDE
CSV files of street crime data from 2018 - 2020 (obtained from UK police dataset, Derbyshire Constabularly)

## Usage

1. place CSV files in folder. To perform analysis this requires the user to have files named "street_Ncsv" where N is year
